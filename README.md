## 🚀 Installation Guide for Dev-C++

### What is Dev-C++?
Dev-C++ is a free, open-source Integrated Development Environment (IDE) for C/C++ programming. It includes MinGW (Minimalist GNU for Windows) compiler.

### Prerequisites
- Windows 7/8/10/11
- Administrator access (for installation)
- Minimum 500MB free disk space

### Step-by-Step Installation

#### Method 1: Installing Dev-C++

**Step 1: Download Dev-C++**
1. Go to the official Dev-C++ download page: [https://sourceforge.net/projects/orwelldevcpp/](https://sourceforge.net/projects/orwelldevcpp/)
2. Click on the **"Download"** button
3. Choose the latest version (recommended: Dev-C++ 5.11)

**Step 2: Run the Installer**
1. Locate the downloaded file (e.g., `devcpp-5.11-TDM-GCC-4.9.2-Setup.exe`)
2. Double-click to run the installer
3. If prompted by Windows Defender, click **"More info"** then **"Run anyway"**

**Step 3: Installation Wizard**
1. **Choose Language**: Select **English** (or your preferred language)
2. Click **"I Agree"** to accept the license agreement
3. **Select Components**: Keep all components selected
   - ✓ Dev-C++ Core Files
   - ✓ MinGW Compiler
   - ✓ Additional Languages (optional)
4. **Choose Install Location**: 
   - Default: `C:\Dev-Cpp`
   - Click **"Browse"** if you want to change location
5. Click **"Install"** and wait for installation to complete
6. Click **"Finish"** to exit the wizard

**Step 4: First Launch Configuration**
1. When Dev-C++ opens for the first time:
   - Select **"English"** as language
   - Choose your **"Theme"** (Dark/Light)
   - Set **"Font Size"** (recommended: 10-12)
2. Click **"Next"** to complete setup
